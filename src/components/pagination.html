Pagination

목록·테이블의 페이지를 이동하는 컴포넌트입니다. nav · ul 시맨틱 구조와 aria-current="page"로 접근성을 지원합니다.

기본

이전·다음 버튼과 페이지 번호로 구성합니다. 현재 페이지는 is-active와 aria-current="page"를 함께 사용합니다.

미니멀

pagination_minimal로 테두리 없는 가벼운 스타일을 적용합니다. 카드·모달 하단에 적합합니다.

크기

pagination_sm · pagination_lg로 항목 스케일을 조절합니다.

둥근

pagination_round로 pill 형태의 페이지 버튼을 적용합니다.

페이지 생략

페이지가 많을 때 pagination_ellipsis로 중간 구간을 생략합니다. 생략 표시는 장식용이므로 aria-hidden="true"를 사용합니다.

pagination_simple로 이전·다음과 현재/전체 페이지만 표시합니다. 모바일·슬라이더형 목록에 적합합니다.

툴바 조합

pagination_total · pagination_jumper와 함께 테이블 하단 툴바를 구성합니다. pagination_wrap으로 좌우 정렬합니다.

정렬

pagination_align-center · pagination_align-end로 가운데·우측 정렬합니다. 기본값은 좌측 정렬입니다.

상태

첫·마지막 페이지에서는 이전·다음 버튼을 disabled로 비활성화합니다.
