Offcanvas

화면 가장자리에서 슬라이드되어 나타나는 패널 컴포넌트입니다. 모바일 내비게이션·필터·설정·장바구니 등 보조 UI에 사용하며, role="dialog" · aria-modal로 접근성을 보장합니다.

기본

data-offcanvas-trigger로 패널을 열고, 백드롭·닫기 버튼·Esc로 닫습니다. 기본 위치는 왼쪽(offcanvas_placement-start)입니다.

위치

offcanvas_placement-start · -end · -top · -bottom으로 슬라이드 방향을 지정합니다.

기본 동작은 초기 로드 시 모두 닫힌 상태이며, 버튼 클릭 이벤트로만 열립니다.

크기

offcanvas_sm · offcanvas_lg으로 패널 너비(또는 상·하 배치 시 높이)를 조정합니다.

백드롭 없음

data-offcanvas-backdrop="false"로 어두운 배경을 숨깁니다. 본문과 동시에 상호작용할 수 있습니다.

헤더·푸터

offcanvas_header · offcanvas_body · offcanvas_footer로 영역을 나눕니다. 푸터에 액션 버튼을 배치합니다.

메뉴

Menu 컴포넌트와 조합해 모바일 내비게이션 패널을 구성합니다.

로드 시 자동 열기 (옵션)

data-offcanvas-open-on-load="true"를 지정하면 페이지 로드 후 해당 Offcanvas를 자동으로 열 수 있습니다. 기본값은 비활성입니다.

아래 예시는 옵션 동작 확인용입니다.

메뉴

Offcanvas 패널 본문입니다. 스크롤이 필요한 긴 콘텐츠도 offcanvas_body 안에서 처리됩니다.

백드롭을 클릭하거나 닫기 버튼, Esc 키로 패널을 닫을 수 있습니다.

왼쪽 패널

화면 왼쪽에서 슬라이드됩니다.

오른쪽 패널

장바구니·알림 등 보조 패널에 자주 사용합니다.

상단 패널

검색·공지 배너 등 상단에서 내려오는 패널입니다.

하단 패널

모바일 액션 시트·공유 패널 등에 적합합니다.

Small

offcanvas_sm — 좁은 보조 패널

Default

기본 너비 20rem

Large

offcanvas_lg — 넓은 설정·상세 패널

백드롭 없음

본문을 가리지 않고 패널만 표시합니다. 닫기 버튼이나 Esc로 닫으세요.

자동 열기 옵션

이 패널은 data-offcanvas-open-on-load="true" 예시입니다.

실서비스에서는 온보딩, 공지, 필수 확인 패널 등에 선택적으로 사용할 수 있습니다.

마크업

트리거 버튼과 Offcanvas 루트를 id · aria-controls로 연결합니다. 기본은 닫힌 상태(hidden)이며, 클릭 이벤트로 열립니다.


<!-- 트리거 -->
<button type="button" data-offcanvas-trigger="#oc-1"
  aria-controls="oc-1" aria-expanded="false">
  메뉴 열기
</button>

<!-- Offcanvas -->
<div class="offcanvas" id="oc-1" data-offcanvas
  role="dialog" aria-modal="true" aria-labelledby="oc-1-title"
  tabindex="-1" hidden>
  <div class="offcanvas_backdrop" data-offcanvas-close aria-hidden="true"></div>
  <div class="offcanvas_panel offcanvas_placement-start">
    <div class="offcanvas_header">
      <h2 class="offcanvas_title" id="oc-1-title">제목</h2>
      <button type="button" class="offcanvas_close"
        data-offcanvas-close aria-label="닫기">…</button>
    </div>
    <div class="offcanvas_body">본문</div>
    <div class="offcanvas_footer">
      <button type="button" class="btn btn_ghost" data-offcanvas-close>취소</button>
      <button type="button" class="btn btn_filled color_primary" data-offcanvas-close>확인</button>
    </div>
  </div>
</div>

<!-- 옵션: 페이지 로드 시 자동 열기 -->
<div class="offcanvas" id="oc-auto" data-offcanvas
  data-offcanvas-open-on-load="true" …>…</div>

<!-- 옵션: 백드롭 없음 -->
<div class="offcanvas" id="oc-plain" data-offcanvas
  data-offcanvas-backdrop="false" …>…</div>
