Popover

클릭·호버로 트리거 옆에 떠 있는 정보 패널 컴포넌트입니다. 짧은 설명·확인·간단한 폼 등 Dropdown보다 풍부한 콘텐츠를 표시할 때 사용하며, aria-haspopup="dialog" · role="dialog"로 접근성을 보장합니다.

기본

popover · popover_trigger · popover_panel으로 트리거와 패널을 구성합니다. data-popover로 클릭 시 열고 닫습니다.

트리거를 클릭하면 이 패널이 표시됩니다. 바깥을 클릭하거나 Esc로 닫을 수 있습니다.

열린 상태

is-open 클래스와 aria-expanded="true"로 패널이 열린 상태를 표현합니다.

정적 데모용 열린 상태입니다.

크기

popover_sm · popover(기본) · popover_lg로 패딩·글자 크기·패널 너비를 조절합니다.

작은 패널입니다. 짧은 안내에 적합합니다.

기본 크기 패널입니다.

큰 패널입니다. 본문이 길거나 폼·목록을 담을 때 사용합니다.

제목

popover_title로 패널 상단에 제목을 표시합니다. aria-labelledby로 패널과 연결합니다.

배송 안내

평일 오후 2시 이전 주문은 당일 출고됩니다. 제주·도서 산간 지역은 1~2일 추가 소요될 수 있습니다.

푸터

popover_footer에 확인·취소 등 액션 버튼을 배치합니다.

항목 삭제

이 작업은 되돌릴 수 없습니다. 계속하시겠습니까?

호버 트리거

data-popover-trigger="hover"로 마우스 오버·포커스 시 패널을 표시합니다. 짧은 설명·용어 정의에 적합합니다.

결제 시 구매자가 상품 수령을 확인할 때까지 대금을 안전하게 보관하는 결제 방식입니다. 서비스를 이용할 수 있습니다.

트리거 유형

Button · Link · 아이콘 버튼을 트리거로 사용할 수 있습니다.

버튼 트리거 예시입니다.

링크 스타일 트리거입니다.

아이콘 전용 트리거입니다.

폼

Input · Form Layout 컴포넌트와 조합해 간단한 입력 UI를 패널 안에 배치합니다.

빠른 메모

화살표 없음

popover_no-arrow로 화살표를 숨깁니다.

화살표가 표시되지 않는 패널입니다.

마크업

트리거와 패널을 aria-controls · aria-labelledby로 연결합니다. 기본은 닫힌 상태(hidden)이며, 클릭 시 열립니다.


<!-- 기본 (클릭) -->
<div class="popover" data-popover>
  <button type="button" class="btn btn_outline popover_trigger"
    aria-haspopup="dialog" aria-expanded="false" aria-controls="popover-1">
    <span class="btn_label">Popover 열기</span>
  </button>
  <div class="popover_panel" id="popover-1" role="dialog"
    aria-labelledby="popover-1-title" hidden>
    <span class="popover_arrow" aria-hidden="true"></span>
    <div class="popover_title" id="popover-1-title">제목</div>
    <div class="popover_body">본문</div>
    <div class="popover_footer">
      <button type="button" class="btn btn_ghost btn_sm" data-popover-close>취소</button>
      <button type="button" class="btn btn_filled color_primary btn_sm" data-popover-close>확인</button>
    </div>
  </div>
</div>

<!-- 호버 트리거 -->
<div class="popover" data-popover data-popover-trigger="hover">…</div>

<!-- 배치 -->
<div class="popover popover_placement-top" data-popover>…</div>

클래스 · 속성

이름 설명

popover_panel플로팅 패널 (role="dialog")
popover_arrow화살표 (장식, aria-hidden="true")
popover_title패널 제목 (aria-labelledby 대상)
popover_body패널 본문
popover_footer하단 액션 영역 (선택)
popover_sm · popover_lg패널 크기 변형
popover_no-arrow화살표 숨김
popover_placement-top · popover_placement-top-center트리거 위쪽 배치
popover_placement-left · popover_placement-right트리거 왼쪽 · 오른쪽
popover_placement-end · popover_placement-bottom-center아래 · 오른쪽/가운데 정렬 (기본은 아래 왼쪽)
data-popoverPopover 루트 식별자 (JS 초기화)
data-popover-trigger="hover"호버 · 포커스 트리거 (기본: 클릭)
data-popover-close닫기 버튼 (푸터 등)
is-open열린 상태 (JS가 토글, hidden 해제와 함께)
is-disabled트리거 비활성 (JS 초기화 생략)
hidden닫힌 초기 상태
aria-haspopup="dialog"트리거에 대화상자 팝업 있음 표시
aria-controls · aria-expanded트리거 ↔ 패널 연결 (트리거에 지정)
aria-labelledby · aria-label패널 제목·이름 연결

닫기: 바깥 클릭 · data-popover-close 버튼 · Esc 키. 클릭 모드에서는 트리거를 다시 클릭해도 닫힙니다. 열릴 때 다른 Popover는 자동으로 닫힙니다.
