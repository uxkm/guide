마크업

data-swiper가 있는 .swiper 요소에 Swiper가 초기화됩니다. .carousel은 레이아웃·스타일 래퍼이며, 슬라이드 안에는 card 등 기존 컴포넌트를 배치합니다.


<!-- 슬라이드 자동 재생 (기본 전환) -->
<div class="carousel carousel_demo">
  <div class="swiper" data-swiper
    data-swiper-autoplay data-swiper-delay="4000" data-swiper-loop
    role="region" aria-roledescription="carousel" aria-label="프로모션">
    …
    <button type="button" class="carousel_toggle"
      aria-label="슬라이드 정지" aria-pressed="true">…</button>
</div>

<!-- 페이드 자동 재생 -->
<div class="carousel carousel_demo">
  <div class="swiper" data-swiper
    data-swiper-autoplay data-swiper-delay="4000"
    data-swiper-loop data-swiper-effect="fade"
    role="region" aria-roledescription="carousel" aria-label="프로모션">
    …
  </div>
</div>

이름 설명
carousel루트 래퍼 (디자인 토큰·레이아웃, 너비 100%)
carousel_demo데모 영역 전체 너비(max-width: 100%)
carousel_multislides per view auto 레이아웃 (슬라이드 너비 가변)
carousel_toggle자동 재생 재생·정지 버튼
carousel_dots-outside페이지네이션을 뷰포트 밖에 배치
swiperSwiper 컨테이너 (data-swiper 필수)
swiper-wrapper슬라이드 트랙 (Swiper 표준)
swiper-slide개별 슬라이드 (내부에 card 등 배치)
data-swiperSwiper 초기화 대상
data-swiper-effect전환 효과 (fade, 생략 시 슬라이드)
data-swiper-autoplay자동 재생 (Autoplay)
data-swiper-delay자동 재생 간격(ms, 기본 5000)
data-swiper-loop무한 루프 (Infinite loop)
data-swiper-slides-per-view동시 표시 슬라이드 수 (auto 또는 숫자)
data-swiper-space-between슬라이드 간격(px)
data-swiper-centered활성 슬라이드 중앙 정렬
data-swiper-pagination페이지네이션 유형 (fraction)
is-autoplay-paused사용자가 정지한 자동 재생 상태
