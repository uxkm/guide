Navbar

사이트 상단 내비게이션 바입니다. 브랜드·메뉴·검색·액션 버튼을 한 줄에 배치하며, 좁은 화면에서는 토글로 메뉴를 접습니다.

기본

navbar 브랜드와 navbar_link 메뉴로 구성합니다. is-active와 aria-current="page"로 현재 페이지를 표시합니다.

구조 · 클래스

주요 블록·파트·변형 클래스 요약입니다.

<header class="navbar navbar_sticky">
  <div class="navbar_container">
    <a href="#" class="navbar_brand">Brand</a>
    <button type="button" class="navbar_toggle"
      aria-expanded="false" aria-controls="main-nav" aria-label="메뉴 열기">…</button>
    <div class="navbar_collapse" id="main-nav">
      <nav class="navbar_nav" aria-label="주요 메뉴">
        <ul class="navbar_list">
          <li class="navbar_item">
            <a href="#" class="navbar_link is-active" aria-current="page">홈</a>
          </li>
        </ul>
      </nav>
      <div class="navbar_search"><input class="input input_sm" type="search"></div>
      <div class="navbar_actions">…</div>
    </div>
  </div>
</header>
클래스 설명
navbar루트 블록
navbar_container내부 정렬·최대 너비
navbar_brand브랜드·로고 링크
navbar_toggle모바일 메뉴 토글 버튼
navbar_collapse접히는 메뉴 영역
navbar_nav · navbar_list내비게이션 목록
navbar_link메뉴 링크 · is-active
navbar_search검색 필드 래퍼
navbar_actions우측 액션 영역
navbar_sm · navbar_lg높이 변형
navbar_sticky상단 고정
navbar_borderless하단 테두리 제거
navbar_darksurface 배경
